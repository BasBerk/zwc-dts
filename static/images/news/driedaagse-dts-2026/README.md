# Driedaagse DTS 2026 Images

This directory contains images for the Driedaagse DTS event landing page.

## Required Images:
- **hero.jpg** - Main hero image for the event (recommended: 1200x600px)
- **tijdrit.jpg** - Image for the time trial section  
- **afvalkoers.jpg** - Image for the elimination race section
- **criterium.jpg** - Image for the criterium section

## Image Guidelines:
- Format: JPG or PNG
- Recommended width: 1200px for hero, 800px for section images
- Optimize for web (keep file sizes under 200KB)
- Alt text and captions should be descriptive for accessibility

## Usage in Content:
Images are referenced using the newsimage shortcode:
```
{{< newsimage src="/images/news/driedaagse-dts-2026/hero.jpg" alt="Description" caption="Caption text" >}}
```
